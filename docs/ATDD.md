# 智考雲 — 驗收測試驅動開發文件 (ATDD - Acceptance Test-Driven Development)

**文件版本**：v1.0
**建立時間**：2026/05/05
**專案名稱**：智考雲 — 企業員工線上考核平台

---

## 1. 文件說明

本文件定義智考雲平台各功能模組的驗收測試案例。每個測試案例包含前置條件、操作步驟與預期結果，用於確認功能是否符合 PRD 需求規格。

### 測試優先級定義
- **P0 — 關鍵路徑**：影響核心考試流程，阻斷使用者主要任務
- **P1 — 重要功能**：影響管理效率或資料正確性
- **P2 — 一般功能**：輔助性功能，不影響核心流程
